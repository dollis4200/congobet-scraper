Guide de configuration

1. Créez un repo GitHub congobet-scraper et poussez-y le code du scraper
2. GitHub Actions scrape automatiquement toutes les 5 min et commit les JSON dans data/
3. Entrez l'URL base raw du repo dans le champ ci-dessus
4. Cliquez ▶ Démarrer — les données se chargent automatiquement
⚠ Si le repo est privé, le fetch direct HTML→GitHub ne fonctionnera pas (CORS). Rendez le repo public ou utilisez un proxy.
